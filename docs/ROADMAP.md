# Roadmap

Поточні цілі проекту та ідеї для розвитку.

## Короткострокові цілі

- додати пошук по контенту
- покрити ключові компоненти тестами
- оновити залежності React/Vite/Supabase
- оптимізувати документацію

## Середньострокові плани

- розширити особистий кабінет
- покращити PWA / offline досвід
- додати мультимовність (українська + англійська)
- покращити SEO для великих сторінок

## Ідеї для майбутніх фіч

- персоналізовані поради для господарств
- інтеграція з калькуляторами кормів
- система нотаток для власників кролів
- спільнота або форум для користувачів
