# Рекомендації для ветеринарного контенту

Дата порівняння: 2026-07-02

## Поточний стан

Проєкт вже містить великий набір ветконтенту на сторінках:

- симптоми
- вакцинація
- пропойка
- паразити
- хвороби
- послід
- контроль ваги
- препарати
- сумісність препаратів
- знеболення
- калькулятор дозування
- лікування
- перша допомога
- отруєння
- некропсія
- кастрація та стерилізація
- лабораторна діагностика
- зоонози

Також є розділи про сезонні загрози та ветеринарні маніпуляції.

## Що варто додати

### 1. `Антибіотикотерапія`

- опис: відповідальна антибіотикотерапія для кролів, вибір препарату, протипоказання, резистентність
- чому: сучасна ветпрактика ставить акцент на безпечному застосуванні антибіотиків, особливо в господарствах із забоєм і продажем продукції
- можливі ключові слова: антибіотики, резистентність, withdrawal, період виведення, ветеринарний нагляд

### 2. `Премедикація, анестезія та післяопераційний догляд`

- опис: підготовка до операцій, безпечні анестетики, моніторинг, відновлення
- чому: існуючі теми про кастрацію та знеболення потребують доповнення конкретною анестезіологією
- можливі ключові слова: анестезія кролик, премедикація, післяопераційний догляд, моніторинг, ускладнення

### 3. `Журнал лікувань та документація`

- опис: як вести записи вакцинацій, дегельмінтизації, лікувань, звернень до ветлікаря
- чому: важливо не лише знати лікування, а й документувати його для контролю стада та відповідності регуляціям
- можливі ключові слова: журнал лікувань, ветеринарна документація, записи вакцинацій, історія хвороб

### 4. `Дієтична терапія`

- опис: кормова підтримка при шлунково-кишкових захворюваннях, реабілітація після лікування, лікувальний раціон
- чому: багато хвороб у кролів мають харчовий компонент, а раціон є ключем до успіху лікування
- можливі ключові слова: лікувальний раціон кролів, дієта при ентериті, харчування після хвороби, підтримка імунітету

## Рекомендації щодо реалізації

1. Додати нові картки у `src/data/sectionCards.ts` під групою `Здоров'я` або `Ветеринарні маніпуляції`.
2. Створити окремі сторінки з практичними алгоритмами:
   - `коли викликати ветеринара`
   - `як вести щоденний журнал стану`
   - `покрокові дії при підозрі на інфекцію`
3. Розширити існуючі сторінки:
   - `Пропойка` — додати підрозділ про відповідальне застосування препаратів
   - `Паразити` — додати профілактику без хімії та критерії, коли обробляти
   - `Перша допомога` — структурувати як «симптом → дія → коли до ветлікаря».

## Очікуваний результат

- проєкт стає ближчим до клінічного довідника для кролівників
- підвищується довіра від ветеринарів та досвідчених фермерів
- зростає ймовірність кращого ранжування за темами, що пов’язані з безпекою та офіційними протоколами
